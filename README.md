## 🎮 Usage guide

The script provides two main features: Aimbot and Invisibility.

### Aimbot usage
The aimbot function locks the view onto enemy targets. It calculates the distance and angle between your character and the target coordinate. 
- Press the default activation key to enable targeting.
- Adjust the sensitivity setting in the configuration file to change how fast the crosshair moves to a target.
- Keep the crosshair near the target to ensure the function triggers reliably.

### Invisibility usage
The invisibility function enables concealment-style mechanics. It manipulates game data buffers to change your visibility state.
- Enable this feature via the menu interface. 
- The software maintains this state as long as the process remains active.
- Disable this feature before you leave the game session.

## 🔧 Troubleshooting common issues

If the tool does not work, check these items:

- Ensure the game runs in windowed mode. Fullscreen mode prevents the overlay from drawing correctly.
- Verify your screen resolution matches the configuration settings. Mismatched resolutions cause errors in target calculation.
- Check the version of the game versus the version of the tool. Updates to the game client often break compatibility.
- Ensure no other overlay software runs simultaneously. Applications like Discord or game launchers often conflict with input redirection.
- Restart the application if the functions stop responding.

## 📝 Configuration settings

You can modify the behavior of the script by editing the settings.ini file. Open this file with Notepad.

- sensitivity: Change this value to 1-10 to define how fast the aimbot tracks targets.
- toggle_key: Change the letter to any key on your keyboard to toggle features.
- log_level: Set to 0 to stop debug messages from appearing in the console.

Save the file and restart the tool to apply changes. Do not delete the ini file, as the program will fail to launch without it. If you break the file, delete it to let the program generate a fresh copy on the next launch.
